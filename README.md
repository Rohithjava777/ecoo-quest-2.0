# 🌍 ECO QUEST — Planetary Restoration Unit

A futuristic **terminal-style web dashboard** designed to gamify environmental actions.
Users complete eco-missions, upload proof, and track their impact on planetary restoration.

---

## 🚀 Features

* ⏱️ **Live Clock (12-hour format)**
* 🎮 **Gamified Missions System**
* 📂 **File Upload for Task Verification**
* 📊 **Dynamic Progress & EXP System**
* 🌊 **Ocean Intelligence Facts Panel**
* 🌍 **Interactive Earth View (Modal UI)**
* 📈 **Environmental Stats Dashboard**
* 📢 **Share Impact Simulation**

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Terminal UI + Animations
* **JavaScript (Vanilla)** – Interactivity & Logic

## 🎯 How It Works

* Users complete **missions** by uploading files

* Each task increases **EXP (Experience Points)**

* EXP updates:

  * 📊 Progress Bar
  * 🌍 Earth Recovery Percentage
  * 🧠 System Logs

* Upon reaching threshold:

  * 🆙 Level increases
  * 🏆 Rank updates
